# **App Name**: ExaminatorAI

## Core Features:

- Exam setup: Define exam title, instructions, duration and questions.
- Question presentation: Present questions one at a time with clear options and input fields.
- Time tracking: Implement a countdown timer and auto-submission on timeout.
- AI Essay grading: Grade essay questions by comparing student answer semantically to a model answer, using an LLM as a tool to judge coverage, validity and coherence.
- Automatic grade calculation: Calculate total score and percentage upon exam completion.
- Result display: Show final grade, percentage, and a summary of essay performance.
- Multi part assessment: Provides different type of assessment options.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to evoke trust and knowledge.
- Background color: Light gray (#F5F5F5) for a clean and distraction-free environment.
- Accent color: Teal (#26A69A) to highlight important actions and notifications.
- Body font: 'Inter', a sans-serif font, for readability.
- Headline font: 'Space Grotesk', a sans-serif font, for a clear, technological feel.
- Use clear and professional icons to guide the user through the exam process.
- Simple and responsive design optimized for both desktop and mobile devices.
